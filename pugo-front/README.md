## Getting Started

First, run the development server:

```bash install
npm install
# or
yarn
```

```bash local dev
npm run dev
# or
yarn dev
```

```bash build
npm run build
# or
yarn build
```

---

```bash ngrok
run ngrok local
# and then
ngrok http http://localhost:8080
```

```bash localtunnel
lt --port 8080
```

---

Open [http://localhost:8080](http://localhost:8080) with your browser to see the result.
